Developer Guide to the BPEL Designer

Contributed by Bob May, maintained by Irina Filippova
December 2007

This document describes the features of the BPEL Designer, which is part of the SOA pack of NetBeans IDE 6.0. It explains basic editing, compilation, deployment and test running of BPEL processes.
